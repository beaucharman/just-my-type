3 Text Level Elements
3.11 Abbreviation <abbr>

The HTML <abbr> element represents an abbreviation and optionally provides a full description for it. If present, the title attribute must contain this full description and nothing else

Sample: dr

The HTML <a> element defines a hyperlink, the named target destination for a hyperlink, or both.

3.5 Citation <cite>

The HTML <cite> element contains the title of a work, such as a book, song, movie, TV show, sculpture, etc.

Sample: This is quotation source

3.7 Code <code>

The HTML <code> element represents a fragment of computer code. By default, it is displayed in the browser's default monospace font.

Sample: document.write("Hello world");

3.6 Definition <dfn>

The HTML <dfn> element represents the defining instance of a term.

Sample: This is definition

3.2 Emphasis <em>

The HTML <em> element marks text that has stress emphasis. The <em> element can be nested, with each level of nesting indicating a greater degree of emphasis.

Sample: This is emphasied text.

3.14 Insert and Delete <ins> <del>

The HTML <ins> element represents a range of text that has been added to a document.

The HTML <del> element represents a range of text that has been deleted from a document. This element is often (but need not be) rendered with strike-through text.

Sample:

This information was just added.
While this information is no longer valid.

3.9 Keyboard <kbd>

The HTML <kbd> Element represents user input and produces an inline element displayed in the browser's default monotype font.

Sample: press Command + Q to close this window. ESC

3.12 Quotation <q>

The HTML <q> element indicates that the enclosed text is a short inline quotation. This element is intended for short quotations that don't require paragraph breaks, for long quotations use <blockquote> element.

Sample: This is sample quotation

3.8 Sample Text Element <samp>

The HTML <samp> produces an inline element displayed in the browser's default monotype font (Lucida Console). This element was intended to identify sample output from a computer program.

Sample: Error: no such file or directory

3.4 Small <small>

The HTML <small> element makes the text font size one size smaller (for example, from large to medium, or from small to x-small) down to the browser's minimum font size. In HTML5, this element is repurposed to represent side-comments and small print, including copyright and legal text, independent of its styled presentation.

Sample: Small

3.3 Strong <strong>

The HTML <strong> element gives text strong importance, and is typically displayed in bold.

Sample: This is text with stronger emphasis

3.13 Subscript and Superscript <sub> <sup>

The HTML <sub> element defines a span of text that should be displayed, for typographic reasons, lower, and often smaller, than the main span of text.

The HTML <sup> element defines a span of text that should be displayed, for typographic reasons, higher, and often smaller, than the main span of text.

Sample:

H2O

E = mc2

The 14th of January

3.10 Variable <var>

The HTML <var> element represents a variable in a mathematical expression or a programming context.

Sample: counter
